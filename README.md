# EMPLOYE-SUREVERY-ANALYSIS

# Employee Survey Analysis using PostgreSQL & Power BI

## Overview

Employee Survey Analysis is a data analytics project designed to evaluate employee satisfaction, engagement, and organizational performance using SQL and Power BI. The project utilizes HR survey response data to identify trends across departments, employee roles, and engagement factors that influence workplace satisfaction.

The analysis was performed using PostgreSQL for data storage, cleaning, and querying, while Power BI was used to create interactive visualizations and dashboards. Through department-wise analysis, role-based satisfaction assessment, and employee engagement insights, the project demonstrates how data-driven decision-making can help organizations improve employee experience and workplace effectiveness.

## Objectives

* Analyze employee satisfaction across different departments.
* Compare satisfaction levels among Directors, Managers, Supervisors, and Staff.
* Identify key employee engagement factors from survey responses.
* Detect areas of concern that may impact employee experience.
* Support HR decision-making through data-driven insights.

## Tools & Technologies

* PostgreSQL
* pgAdmin
* SQL (CTE, Views, Window Functions)
* Microsoft Power BI
* Microsoft Excel

## Key Insights

* Employee satisfaction varies significantly across departments.
* Leadership roles generally report higher satisfaction scores.
* Employee engagement is influenced by communication, support, and workplace culture.
* Survey analysis helps identify strengths and improvement opportunities within the organization.

## Project Outcome

This project demonstrates the practical application of SQL and Power BI in HR analytics, transforming raw survey data into actionable business insights that can support employee engagement initiatives and organizational improvement strategies.
